FRONTEND CANTIK - SISTEM ADMIN GYM

Cara pasang:
1. Upload / replace folder frontend di repo GitHub kamu dengan folder frontend dari ZIP ini.
2. Buka frontend/config.js.
3. Ganti SCRIPT_URL dengan URL Web App Apps Script yang berakhiran /exec.
4. Commit changes.
5. Tunggu GitHub Pages update 1-2 menit.
6. Buka https://hantugur.github.io/SISTEM-GYM/frontend/

Catatan:
- ENABLE_LOCAL_SCRIPT_URL_SETTING: false artinya pegawai tidak lihat kotak URL backend.
- Kalau mau debugging, ubah sementara jadi true.
- Backend tetap pakai Apps Script yang sudah kamu deploy.
